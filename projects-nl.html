Projecten

project 1: rekenmachine

Dit is een project dat ik heb gemaakt met behulp van PHP, HTML en CSS. Het is een eenvoudige rekenmachine die in de webbrowser draait. De rekenmachine kan basisbewerkingen uitvoeren, zoals optellen, aftrekken, vermenigvuldigen en delen. De rekenmachine kan ook een foutmelding geven wanneer je probeert te delen door 0 of een letter in te voeren in plaats van een cijfer.





project 2: steen papier schaar

Dit is een project dat ik met een klasgenoot moest maken voor ons groepsproject. Het is een eenvoudig steen, papier, schaar spel, gemaakt met PHP, HTML en CSS. Je kunt kiezen of je tegen een andere speler (op hetzelfde apparaat) of tegen de computer wil spelen.





Project 3: Galgje

Dit project is ook het resultaat van een groepsopdracht met mij en mijn klasgenoot. We moesten een volledig werkend galgje spel maken met behulp van PHP, HTML en CSS. Het spel biedt de mogelijkheid om je eigen woord te kiezen of de computer een willekeurig woord te laten kiezen uit een array. Je hebt een beperkt aantal pogingen om het woord te raden. Lukt het niet, dan verlies je het spel.





Project 4: sql database

Dit project bestaat uit een reeks databases gemaakt in MySQL. De eerste database is een eenvoudige database met de planeten in ons zonnestelsel en de gegevens van iedere planeet. De tweede database is iets complexer, omdat deze voor een fictieve streamingdienst genaamd "Netland" is. Deze database bevat tabellen voor films en de gegevens van iedere film. De derde database is voor dezelfde streamingdienst, maar bevat deze keer tabellen voor series en alle bijbehorende gegevens.





Project 5: netland administrator website

In dit project moest ik een administrator paneel maken voor de eerder genoemde Netland-database. Het adminpaneel biedt de mogelijkheid om films en series te sorteren op titel (a-z en z-a) of op beoordeling (hoog naar of laag en laag naar hoog). Je kunt ook rechtstreeks vanuit het adminpaneel nieuwe films en series aan de database toevoegen. Bestaande gegevens bewerken is ook mogelijk vanuit de detailpagina, je komt hier door op de titel van de film of serie te klikken. Het volledige adminpaneel is alleen toegankelijk na het inloggen met de juiste inloggegevens. Het adminpaneel is gemaakt met behulp van PHP, HTML, CSS en PDO voor de databaseverbinding.
