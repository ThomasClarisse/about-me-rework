# About Me Rework

This is a personal website project created to showcase my skills and projects. The website is built using HTML, CSS, and JavaScript, and it features a responsive design that works well on both desktop and mobile devices.

## Features

- Home page with a brief introduction and a dropdown menu with links to other sections
- Option to use the webite in English or Dutch
- Projects page showcasing various projects with descriptions and images
- Information page with details about my background and skills
- Contact page with a form to reach out to me
- "Back to top" button for easy navigation
- Links to my GitHub repository for the project

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Use

The website will have it's own domain soon but in the meantime you have to download the files and open the `index.html` file in your web browser to view the website.
