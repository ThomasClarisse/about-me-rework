# About Me Rework

Dit is een persoonlijk websiteproject, gemaakt om mijn vaardigheden en projecten te laten zien. De website is gebouwd met HTML, CSS en JavaScript en heeft een responsief ontwerp dat goed werkt op zowel desktop als mobiele apparaten.

## Functies

- Startpagina met een korte introductie en een dropdownmenu met links naar andere secties
- Optie om de website in het Engels of Nederlands te gebruiken
- Projectenpagina met verschillende projecten, beschrijvingen en afbeeldingen
- Informatiepagina met details over mijn achtergrond en vaardigheden
- Contactpagina met een formulier om contact met mij op te nemen
- "Terug naar boven" knop voor gemakkelijke navigatie
- Links naar mijn GitHub-repository voor het project

## Hoe te gebruiken

De website zal binnenkort een eigen domein hebben, maar in de tussentijd moet je de bestanden downloaden en het `index-nl.html` bestand openen in je webbrowser om de website te bekijken.
